Right Content Area

This is the content that will be displayed on the right side of the screen when you click on the "Contact" or "About" links.

To create this layout, we used CSS to position the sidebar on the left side of the screen and give it a fixed width. We also used CSS to give the content area a left margin of the same width as the sidebar, so it appears next to the sidebar.

To make the links work, we used anchor tags with the href attribute set to the IDs of the corresponding sections of the content area. For example, the "Contact" link has an href of "#contact", which matches the ID of the contact section of the content area.
